# Frontend

## Overview

This directory contains the React-based web interface for Quantum Forge, providing a dashboard for managing simulations, visualizing results, and monitoring quantum computing resources.

**⚠️ NOTE: This directory and all its contents are currently under active development. Features and APIs are subject to change.**

## Structure

```
frontend/
├── src/                 # Source code
│   ├── components/      # Reusable UI components
│   ├── pages/           # Page components
│   ├── services/        # API services and QPU integration
│   ├── hooks/           # Custom React hooks
│   ├── context/         # React context providers
│   ├── layout/          # Layout components
│   ├── icons/           # SVG icons
│   └── utils/           # Utility functions
├── public/              # Static assets
├── docs/                # Frontend-specific documentation
└── scripts/             # Frontend build scripts
```

## Technology Stack

- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Testing**: Jest + React Testing Library
- **Build Tool**: Vite
