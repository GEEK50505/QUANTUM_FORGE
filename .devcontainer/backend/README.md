# Backend

## Overview

This directory contains the core simulation backend for Quantum Forge, including all classical, quantum, and hybrid simulation modules.

**⚠️ NOTE: This directory and all its contents are currently under active development. Features and APIs are subject to change.**

## Structure

```
backend/
├── simulation/          # Main simulation modules
│   ├── classical_sim/   # Classical molecular dynamics simulator
│   ├── quantum_kernel/  # Quantum mechanical solver
│   ├── hybrid_pipeline/ # Hybrid simulation orchestrator
│   └── utils/          # Visualization and analysis tools
└── db/                  # Database and persistence layer
```
