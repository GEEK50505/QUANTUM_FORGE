# Development Tools

This directory contains personal development environment tools and utilities that are not part of the main QUANTUM_FORGE platform.

## Contents

### OpenRouter Integration (`openrouter/`)
Personal development environment setup for enhanced coding experience:
- OpenRouter API integration
- Key management and rotation
- Development environment setup

### Scripts (`scripts/`)
Development workflow automation tools:
- Key testing and management
- Environment setup
- Development session initialization

## Important Note
These tools are for developer workflow enhancement only and are not part of the QUANTUM_FORGE platform functionality. They should not be imported or used by the platform code.
